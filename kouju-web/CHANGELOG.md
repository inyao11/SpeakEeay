# 更新日志

所有项目的显著变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-04-18

### ✨ 新增功能

#### 核心功能
- 🎯 **项目初始化** - SpeakEasy English 口语学习平台正式发布
- ⏱️ **30分钟学习计时器** - 专注学习模式，完成自动打卡
- 📅 **打卡系统** - 连续打卡天数统计，培养学习习惯
- 📊 **进度可视化** - 学习进度条实时显示

#### 口语学习
- 🗣️ **8大高频场景** - 日常问候、机场出行、餐厅点餐、购物砍价、职场沟通、校园交流、兴趣爱好、健康医疗
- 📝 **64句实用口语** - 每场景8句，覆盖日常生活各个方面
- 🔊 **即时发音** - 点击即可听到标准美式/英式发音
- 🎤 **跟读录音** - 支持浏览器语音识别跟读练习
- 🤖 **AI 智能评分** - 发音准确度、流利度、重音节奏三维度评分

#### 显示控制（特色功能）
- 📄 **显示原文模式** - 仅展示英文句子，适合进阶学习者自测
- 📝 **显示注释模式** - 英文+中文翻译+音标+学习注释
- 🙈 **隐藏模式** - 纯听力磨耳朵训练
- 💾 **状态记忆** - 自动保存用户显示偏好

#### 数据管理
- 💾 **本地存储** - 使用 localStorage 永久保存所有数据
- ⭐ **收藏夹** - 收藏重点句子，支持播放和删除
- 📜 **学习历史** - 记录每次练习，含评分和时间
- 📅 **打卡日历** - 可视化展示每月打卡情况
- 📈 **学习统计** - 总时长、句子数、收藏数、连续天数

#### 界面与体验
- 🌙 **深色模式** - 支持浅色/深色主题切换
- 📱 **响应式设计** - 完美适配手机、平板、电脑
- 🎨 **现代 UI** - 使用 Tailwind CSS 构建清爽界面
- ⚡ **流畅动画** - 评分动画、按钮反馈、过渡效果
- 🔔 **Toast 提示** - 操作反馈即时显示

#### 设置功能
- 🌐 **口音切换** - 美式/英式英语可选
- 🐌 **语速调节** - 0.5x - 1.5x 语速控制
- 🔊 **音量控制** - 0% - 100% 音量调节
- 🗑️ **数据清除** - 一键清除所有本地数据

### 🔧 技术实现

#### 前端架构
- **HTML5** - 语义化结构，SEO 友好
- **CSS3 + Tailwind CSS** - 现代响应式样式框架
- **JavaScript ES6+** - 模块化开发，代码清晰
- **Font Awesome** - 丰富的图标库

#### 核心模块
- `storage.js` - 本地存储管理模块
- `speech.js` - Web Speech API 语音功能封装
- `ai-score.js` - 前端智能评分算法
- `data.js` - 口语数据管理与每日更新
- `display.js` - 显示控制逻辑
- `app.js` - 主应用逻辑整合

#### 语音技术
- **Web Speech API** - 浏览器原生语音合成与识别
- **语音合成** - 支持语速、音量、口音调节
- **语音识别** - 实时录音转文字，支持跟读评分

### 🐛 已知问题

- 语音识别在部分移动端浏览器可能受限
- 首次加载语音列表可能需要等待
- 部分旧版浏览器不支持 Web Speech API
